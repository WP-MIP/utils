Welcome to the WP-MIP Forecast Repository

This project contains the information critical to understand the WP-MIP forecast archive.  The forecast datasets themselves should be accessed through the [WP-MIP project website](https://www.wcrp-esmo.org/activities/wp-mip).

Although the forecast dataset is kept as stable as possible, corrections are documented at in the [update issue](https://github.com/WP-MIP/.github/issues/37).
